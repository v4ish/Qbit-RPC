# [Qbit-RPC](https://github.com/v4ish/rpc)
Discord Rich presence for [QBittorrent](https://www.qbittorrent.org/) written in c#

## Features

- Definetly Lightweight 
- Supported os: Windows
- Runs on x86, x64 & ARM32, ARM64 architecture
